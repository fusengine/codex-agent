# Voice & Tone by Sector

## NNG 4 Dimensions

| Dimension | Scale |
|-----------|-------|
| **Formality** | Formal ←—→ Casual |
| **Humor** | Funny ←—→ Serious |
| **Respect** | Respectful ←—→ Irreverent |
| **Enthusiasm** | Enthusiastic ←—→ Matter-of-fact |

## Fintech / Banking

| Dimension | Setting |
|-----------|---------|
| Formality | Formal |
| Humor | Serious |
| Respect | Respectful |
| Enthusiasm | Matter-of-fact |

| Context | Do | Don't |
|---------|-----|-------|
| CTA | "Transfer funds" | "Send money fast!" |
| Success | "Your transfer is complete. Reference: TXN-2847" | "Yay! Money sent!" |
| Error | "Card declined. Check expiration date and try again." | "Uh oh! Something broke." |
| Empty state | "No transactions yet" | "Nothing here... yet! 🎉" |
| Helper text | "Used to verify account ownership" | "So we know who you are" |
