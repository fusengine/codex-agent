# CTA Patterns — 20+ Examples by Context

## Signup / Registration

| Context | CTA |
|---------|-----|
| Free tier | "Start for free" |
| Trial offer | "Start 14-day free trial" |
| No CC required | "Try free — no credit card needed" |
| Quick signup | "Create account in 30 seconds" |
| Social auth | "Continue with Google" |

## Purchase / Upgrade

| Context | CTA |
|---------|-----|
| Direct purchase | "Buy now — $49" |
| Add to cart | "Add to cart" |
| Upgrade plan | "Upgrade to Pro" |
| Confirm order | "Place order — $89.00" |
| Subscribe | "Subscribe and save 40%" |

## Learn More / Explore

| Context | CTA |
|---------|-----|
| Feature page | "See how it works" |
| Demo request | "Book a demo" |
| Documentation | "Read the docs" |
| Case study | "Read the story" |
| Video | "Watch the walkthrough" |

## Action / Submit

| Context | CTA |
|---------|-----|
| Form submit | "Save changes" (not "Submit") |
| File upload | "Upload file" |
| Invite team | "Send invites" |
| Delete confirm | "Delete project" (not "Confirm") |
| Publish | "Publish to live" |

## Recovery / Re-engagement

| Context | CTA |
|---------|-----|
| Password reset | "Reset my password" |
| Reactivate | "Reactivate my account" |
| Empty state | "Create your first project" |
| Abandoned cart | "Continue shopping" |
| Offboarding | "Pause instead of canceling" |

## Rules
- Use specific verbs: "Create", "Send", "Save", "Publish", "Book" — not "Submit", "OK", "Go"
- Include value when possible: "Start free trial" > "Sign up"
- Destructive actions: use the object name — "Delete project" not "Confirm"
- Max 3 words for buttons; use description text for context
- Primary CTA: high contrast, full width on mobile
