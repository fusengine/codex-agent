# UX Copy — SaaS

## Feature Descriptions

### Value-Led (benefit first, feature second)
```
✓ "Ship faster — AI writes your tests as you code"
✓ "Never lose a lead — sync contacts across 50+ tools automatically"
✓ "Cut support tickets by 40% with self-service docs"
✗ "AI-powered code generation with multi-language support"
```

## Pricing Copy

### Plan Names & CTAs
```
Starter:   "Get started free" (no credit card)
Pro:       "Start 14-day free trial"
Enterprise:"Talk to sales"
```

### Pricing Page Framing
```
✓ "Save 40% with annual billing"
✓ "Most popular" badge on recommended plan
✓ "Cancel anytime — no questions asked"
✓ "$49/month, billed annually ($588/yr)"
✗ "Starting at $49" (hides total cost)
```

## Onboarding Flows

### Welcome Message
```
"Welcome to [App], [Name]!
Let's set up your workspace in 3 steps — takes about 5 minutes."
```

### Step Progress
```
"Step 2 of 3 — Connect your tools"
"Almost there! One last thing."
```

### Completion
```
"You're all set! Here's what you can do next:"
[3 concrete next actions, not generic "explore the app"]
```

## Empty States

| Context | Copy |
|---------|------|
| No projects | "No projects yet — create your first one in 30 seconds" |
| No team members | "Invite your team — collaboration is better together" |
| No data yet | "Your dashboard will populate as data comes in" |
| Search no results | 'No results for "X" — try different keywords or browse all' |

## Error & Limit Messages

| Scenario | Message |
|----------|---------|
| Feature locked | "Upgrade to Pro to unlock unlimited exports" |
| Rate limit | "You've hit the hourly limit. Resets in 47 minutes." |
| API error | "Something went wrong on our end. We've been notified — try again in a moment." |
| Workspace limit | "Your plan supports 5 seats. Add more or remove a member." |

## Tone Rules
- Lead with outcomes, not features
- "Free trial" not "trial period"
- Always state what "free" includes (and excludes)
- Error messages must offer a clear next action
