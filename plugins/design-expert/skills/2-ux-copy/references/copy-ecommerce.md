# UX Copy — E-commerce

## Conversion Copy

### Product Descriptions
```
✓ "Free shipping on orders over $50 — arrives by Thursday"
✓ "4.8 ★ (2,341 reviews) — #1 bestseller in Running Shoes"
✓ "Only 3 left in stock"
✗ "This product is a high-quality item available for purchase"
```

### Urgency (ethical — must be true)
```
✓ "Sale ends in 2h 14m"
✓ "12 people are viewing this right now"
✓ "Ships today if ordered within 3 hours"
✗ "Hurry! Limited time offer!" (fake urgency, no specifics)
```

## Social Proof

```
✓ "Worn by 14,000+ runners this season"
✓ "As seen in Wired, TechCrunch, Fast Company"
✓ '"Finally, a bag that fits everything." — Sarah M., verified buyer'
✓ "Rated 4.9/5 by 847 customers"
```

## Cart & Checkout

| Stage | Copy |
|-------|------|
| Add to cart CTA | "Add to bag" / "Add to cart" (not "Buy now" unless instant purchase) |
| Cart empty | "Your cart is empty — browse bestsellers" |
| Order summary | "You're saving $14.00 today" |
| Checkout CTA | "Place order — $89.00" (show total on button) |
| Confirm email | "Order #4821 confirmed — check your email for details" |

## Post-Purchase

```
✓ "Order placed! Estimated delivery: March 22–24"
✓ "We'll email you when your order ships"
✓ "Questions? Reply to this email or chat with us"
```

## Error Messages

| Scenario | Message |
|----------|---------|
| Out of stock | "Sold out — join the waitlist to be notified" |
| Promo code invalid | "Code SAVE20 isn't valid for this cart. Check the terms." |
| Payment failed | "Payment didn't go through. Check your card details or try PayPal." |
| Address unverified | "We couldn't verify this address. Double-check and try again." |

## Tone Rules
- Be specific with numbers (amounts, dates, ratings)
- Use "you/your" — personal and direct
- Show savings prominently, never hide fees
- Urgency copy must be accurate — false scarcity erodes trust
