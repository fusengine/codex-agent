# fuse-design

Expert UI/UX design with shadcn/ui, 21st.dev, and WCAG accessibility.

## Agents

| Agent | Description |
|-------|-------------|
| `design-expert` | Complete UI/UX design |

## Skills

| Skill | Description |
|-------|-------------|
| `design` | APEX design workflow |
| `generating-components` | Production-ready components |
| `designing-systems` | Design tokens, palettes |
| `adding-animations` | Framer Motion |
| `validating-accessibility` | WCAG 2.2 AA compliance |

## Commands

| Command | Description |
|---------|-------------|
| `/design` | UI/UX design workflow |

## MCP Servers

- Context7 (documentation)
- Exa (web search)
- Magic (21st.dev component generation)
- shadcn (component registry)
- Sequential Thinking

## Accessibility Rules

- `aria-label` on interactive elements
- `alt` on images
- Color contrast ratios
- Keyboard navigation
- No emojis as icons
